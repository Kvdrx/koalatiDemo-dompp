# koalatiDemo-dompp


This is a demo of the tests written on the Koalati website with the DOM-PP library


Installing and building DOM-PP with KOALATI 
-------------------------------------------

## Git Clone

```bash
git clone https://github.com/Kvdrx/koalatiDemo-dompp.git
```

## Install

```bash
npm install  
```

## Testing

```bash
npm test
```
